Introduction

API for Web Application Assignment Year 4

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

Register the user

Example request:
curl --request POST \
    "http://localhost/api/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"firstName\": \"sevoo\",
    \"lastName\": \"qlbxwen\",
    \"gender\": \"female\",
    \"email\": \"roberts.velma@example.net\",
    \"phoneNumber\": 9.43317299,
    \"country\": \"ratojqyvmeaal\",
    \"city\": \"smxdiswfkzliiuheittt\",
    \"password\": \"v\\\\0Q}N3T^t$Y!j5\"
}"

const url = new URL(
    "http://localhost/api/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "firstName": "sevoo",
    "lastName": "qlbxwen",
    "gender": "female",
    "email": "roberts.velma@example.net",
    "phoneNumber": 9.43317299,
    "country": "ratojqyvmeaal",
    "city": "smxdiswfkzliiuheittt",
    "password": "v\\0Q}N3T^t$Y!j5"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Request      

POST api/register

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

firstName   string   

Must not be greater than 255 characters. Example: sevoo

lastName   string   

Must not be greater than 255 characters. Example: qlbxwen

gender   string   

Example: female

Must be one of:
  • male
  • female
email   string   

Must be a valid email address. Must not be greater than 50 characters. Example: roberts.velma@example.net

phoneNumber   number   

Example: 9.43317299

country   string   

Must not be greater than 100 characters. Example: ratojqyvmeaal

city   string   

Must not be greater than 100 characters. Example: smxdiswfkzliiuheittt

password   string   

Must match the regex /^.(?=.{3,})(?=.[a-zA-Z])(?=.[0-9])(?=.[\d\x])(?=.[!$#%]).$/. Must be at least 6 characters. Example: v\0Q}N3T^t$Y!j5

profile   string  optional
