Introduction

API for Web Application Assignment Year 4

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

Register the user

Example request:
curl --request POST \
    "http://localhost/api/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"firstName\": \"rcborbbaospo\",
    \"lastName\": \"vfpgh\",
    \"gender\": \"female\",
    \"email\": \"xwaelchi@example.net\",
    \"phoneNumber\": 35491705.8256862,
    \"country\": \"bahvcfnytjwgjntkyiovkgdhh\",
    \"city\": \"mmbpm\",
    \"password\": \"pA~P\\/%\"
}"

const url = new URL(
    "http://localhost/api/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "firstName": "rcborbbaospo",
    "lastName": "vfpgh",
    "gender": "female",
    "email": "xwaelchi@example.net",
    "phoneNumber": 35491705.8256862,
    "country": "bahvcfnytjwgjntkyiovkgdhh",
    "city": "mmbpm",
    "password": "pA~P\/%"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Request      

POST api/register

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

firstName   string   

Must not be greater than 255 characters. Example: rcborbbaospo

lastName   string   

Must not be greater than 255 characters. Example: vfpgh

gender   string   

Example: female

Must be one of:
  • male
  • female
email   string   

Must be a valid email address. Must not be greater than 50 characters. Example: xwaelchi@example.net

phoneNumber   number   

Example: 35491705.825686

country   string   

Must not be greater than 100 characters. Example: bahvcfnytjwgjntkyiovkgdhh

city   string   

Must not be greater than 100 characters. Example: mmbpm

password   string   

Must match the regex /^.(?=.{3,})(?=.[a-zA-Z])(?=.[0-9])(?=.[\d\x])(?=.[!$#%]).$/. Must be at least 6 characters. Example: pA~P/%

profile   string  optional
